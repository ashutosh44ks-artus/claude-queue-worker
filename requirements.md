# Requirements for Claude Queue Worker

- Node.js 22 or higher
- Access to the `claude` CLI tool v2.1+ (must be installed and configured properly)
- `claude` CLI must be logged in

## FAQ

### I have not installed the `claude` CLI tool. How do I do that?
You can install the `claude` CLI tool by following the instructions in the official repository: https://code.claude.com/docs/en/quickstart

### I have installed the `claude` CLI tool, but I am not logged in. How do I log in?
You can log in to the `claude` CLI tool by running the following command in your terminal:
```claude login
```
This will prompt you to enter your credentials and complete the login process.

### How to install Node.js 22 or higher?
You can install Node.js 22 or higher by following the instructions on the official Node.js website: https://nodejs.org/en/download/
